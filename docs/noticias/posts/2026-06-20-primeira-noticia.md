# Primeira Notícia do Nosso Portal

Bem-vindo ao novo portal de notícias estático! 

Este site foi construído utilizando **MkDocs Material** e está integrado ao **Decap CMS** para permitir a publicação e edição de notícias de forma extremamente prática, direto pelo celular ou qualquer navegador web.

## Como funciona?

1. Você acessa a rota `/admin` do seu site.
2. Faz a autenticação.
3. Escreve a sua notícia utilizando um editor visual amigável.
4. Clica em publicar.
5. O CMS gera um arquivo Markdown no GitHub e dispara a pipeline de publicação automática (GitHub Actions).

Pronto! Em poucos minutos, a notícia estará no ar para todos os seus leitores.
